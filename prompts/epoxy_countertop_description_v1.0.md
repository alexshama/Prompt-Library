# Промпт: Продающее описание столешницы из эпоксидной смолы

**Версия:** v1.0  
**Владелец:** alexshama 
**Теги:** копирайтинг, ecommerce, маркетплейсы, описание_товара, мебель, epoxy  
**Цель:** Автоматически генерировать структурированные и продающие описания столешниц из эпоксидной смолы на основе входных данных без выдумывания характеристик.

---

## Шаблон промпта (по фреймворку CLEAR)

### Task
Создай продающее описание для {тип_товара}, чтобы повысить {цель: конверсию/интерес/продажи}.

### Context
Площадка: {маркетплейс/сайт/соцсети}.  
Тон: {дружелюбный/премиальный/экспертный}.  
Стиль: {кратко/эмоционально/информативно}.  
Язык: русский.  
Важно: не выдумывай факты и характеристики — используй только данные из Input.

### Input
Название: {название}  
Материалы: {материалы}  
Размеры: {размеры}  
Дизайн/цвет: {дизайн}  
Преимущества/УТП: {преимущества}  
Целевая аудитория: {аудитория}  
Сценарии использования: {сценарии}  
Ограничения/стоп-слова: {стоп_слова}

### Output
Верни результат в структуре:

1. Заголовок (до {n} символов)
2. Краткий эмоциональный абзац (2–3 предложения)
3. 5–7 преимуществ (маркерный список)
4. Характеристики (список или таблица)
5. Применение (1–3 строки)
6. Призыв к покупке (CTA)

Проверь:
- нет выдуманных фактов
- минимум воды
- текст легко читается
- соответствует заданному тону

---

## Пример входных данных

Тип товара: столешница из эпоксидной смолы  
Цель: увеличить продажи на маркетплейсе  
Площадка: Ozon  
Тон: современный, тёплый, премиальный  
Стиль: кратко и по делу  

Название: River Table  
Материалы: массив дуба + прозрачная эпоксидная смола  
Размеры: 160×80×4 см  
Дизайн: эффект «реки»  
Преимущества: ручная работа, уникальный рисунок, влагостойкость, простой уход  
Аудитория: владельцы квартир, дизайнеры, кафе  
Сценарии: кухня, гостиная, кофейня  
Стоп-слова: самая лучшая, вечная, неубиваемая

---

## Пример результата (сокращённо)

**Заголовок:** Столешница River Table — натуральный дуб и эффект «реки»  

**Описание:**  
Эффектное сочетание массива дуба и прозрачной смолы превращает столешницу в арт-объект. Каждое изделие уникально благодаря природному рисунку древесины.

**Преимущества:**  
• ручная работа  
• уникальный дизайн  
• устойчивость к влаге  
• прочная поверхность  
• лёгкий уход  

**Характеристики:**  
160×80×4 см, дуб + эпоксидная смола  

**CTA:**  
Добавьте в интерьер уникальный акцент — закажите свою столешницу сегодня.

---

## Тест-кейсы

### Тест 1 — минимальные данные
Вход: только название + материалы + размеры  
Ожидание:
- структура соблюдена
- нет выдуманных свойств
- текст остаётся продающим

### Тест 2 — премиальный сегмент
Вход: тон = премиальный, аудитория = дизайнеры интерьеров  
Ожидание:
- акцент на эстетике и уникальности
- без «дёшево/скидка»
- более изысканный стиль

### Тест 3 — ограничения по формулировкам
Вход: заданы стоп-слова  
Ожидание:
- запрещённые слова отсутствуют
- используются только разрешённые формулировки
- преимущества сформулированы корректно
